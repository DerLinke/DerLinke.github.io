DerLinke Repo

Offizielles Software-Repository für Debian & Ubuntu

Über dieses Repository

Willkommen im zentralen Software-Verzeichnis von DerLinke. Hier findest du optimierte Tools für Paketmanagement, System-Updates und Dashboard-Funktionalität.

Verfügbare Pakete

  • FinDeb – Der universelle Package Manager Wrapper.
  • Ultimate-Debian-Updater – Automatisiertes, intelligentes Update-System.
  • MyDash – Minimalistisches CLI-Dashboard für System-Metriken.

Installation

Um dieses Repository zu deinem System hinzuzufügen, führe folgende Befehle aus:

# 1. GPG-Schlüssel hinzufügen
curl -s https://derlinke.github.io/repo/derlinke-repo.gpg.key | gpg --dearmor | sudo tee /usr/share/keyrings/derlinke-repo.gpg > /dev/null

# 2. Repository-Quelle eintragen
echo "deb [signed-by=/usr/share/keyrings/derlinke-repo.gpg] https://derlinke.github.io/repo/stable main" | sudo tee /etc/apt/sources.list.d/derlinke.list

# 3. Paketliste aktualisieren
sudo apt update
